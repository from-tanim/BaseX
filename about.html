About NS-CALC Project

Welcome to My Repository!

This project is designed to help in converting any number system with the decimal quantity on it. It was created with the goal of saving the time and also to make the students helpful in practicing this topic or chapter.

Features

  • Feature 1: You can convert any number to any number system.
  • Feature 2: You can also convert the number with a decimal quantity in it.
  • Feature 3: Further, you also have a copy button in it to help you to copy the result.

Technologies Used

This project leverages the power of:

  • HTML5
  • CSS3
  • JavaScript
  • Json
